Sou Bacharel em Sistemas de Informação, Oracle Certifield Associate Java Programmer 8. Eu adoro estudar, conhecer novas tecnologias e sempre buscar novas técnicas de aperfeiçoamento e boas pratacas. Estou no mercado de TI desde meados de 2012. Desde lá então, trabalhei com diversas tecnologias e frameworks de mercado.

Experiências

UNIPAM

Setembro 2011 até fevereiro 2012. Centro Universitário de Patos de Minas; Desenvolvedor Java voluntário no projeto BIS(Banco de Informações Sociais). Desenvolvimento de um sistema web, parceria entre o UNIPAM e a Prefeitura Municipal de Patos de Minas. Utilizando as tecnologias Java, JSP, JSON, CSS, Javascript, Vraptor, Hibernate, MySQL e SVN. Utilizando Framework ágil Scrum e outras práticas.

PALGER

Março 2012 até Junho de 2014. PALGER LTDA ME; Patos de Minas – MG. Análise e desenvolvimento de Sistemas. Desenvolvimento de sistemas voltados para a web, reuniões com clientes, análise de requisitos, modelagem de banco de dados, deploy de aplicações web. Trabalhando com as tecnologias e frameworks Java, Java Server Faces, Primefaces, Hibernate, Spring, Spring Security, RestEasy, Rmi, Swing, Mysql, CloudComputing utilizando as plataformas PaaS, SaaS e IaaS.

SOFTBOX

Junho 2014 até outubro 2015. Softbox - Soluções Inteligentes em TI; Uberlândia – MG. Análise e desenvolvimento de Sistemas. Adição de novas funcionalidades no sistema, manutenção, adição de testes unitários. Integração de sistemas, utilizando as linguagens Java e PHP. Utilizando Framework ágil Scrum, Integração Contínua(CI), ferramentas de análise de qualidade de código, 80% de cobertura de código. Projeto: http://www.dafiti.com.br/

CTIS

Outubro 2015 até presente. CTIS; Brasília – DF. Desenvolvedor de Sistemas. Adição de novas funcionalidades no sistema, manutenção. Utilizando tecnologias como Java, Angular, Apache Solr, banco de dados Oracle, entre outras. Projeto: Plataforma do governo, CNPQ.
